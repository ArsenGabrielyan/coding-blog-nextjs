# CodeBlog
Advanced Coding Blog Page With Next JS, MongoDB, Firebase and SWR

## Install
```bash
git clone https://github.com/ArsenGabrielyan/coding-blog-nextjs.git
cd coding-blog-nextjs
npm install
npm run dev
```

## Tech
- **Auth:** Next-Auth
- **Real Time Data Fetch:** SWR
- **Storage:** Firebase
- **Database:** MongoDB
- **FrontEnd:** React JS
- **Backend:** Next JS Backend
